## 1. ملخص المشروع

متجر إلكتروني متكامل لبيع منتجات التجميل الرجالية والنسائية، يدعم البيع بالجملة والتجزئة، مع نظام طلب عبر واتساب مباشر، ولوحة تحكم إدارية كاملة، وتصميم متجاوب (RWD) مع انتقالات سينمائية.

---

## 2. التقنيات المستخدمة

| الطبقة | التقنية |
|--------|---------|
| **Frontend** | Next.js 16 (App Router) + TypeScript + Tailwind CSS v4 |
| **Animations** | Framer Motion (AnimatePresence, spring physics, variants) |
| **Smooth Scroll** | Locomotive Scroll v5 (Lenis engine) |
| **Backend** | Next.js API Routes + Server Actions |
| **Database** | Prisma v5 + SQLite |
| **Auth** | NextAuth.js v5 (Credentials + JWT + bcrypt) |
| **Validation** | Zod |
| **Icons** | Lucide React |
| **Notifications** | React Hot Toast |
| **RWD** | Mobile-first, fluid typography (clamp), 4 breakpoints |

## 4. الميزات المنجزة

### 4.1 واجهة المتجر العامة

| الميزة | الوصف |
|--------|-------|
| **Hero سينمائي** | خلفية داكنة مع orbs متحركة، نص متدرج، أزرار CTA، مؤشر تمرير |
| **منتجات مميزة** | Grid متجاوب (1→4 أعمدة)، كروت مع hover zoom + overlay واتساب |
| **قسمين رجالي/نسائي** | صفحات منفصلة مع تصفية حسب القسم |
| **الجملة** | صفحة مخصصة للمنتجات المتاحة للجملة مع سعرين |
| **تفاصيل المنتج** | صفحة كاملة: صورة، اسم، وصف، سعرين، تاغات، زر واتساب |
| **RWD** | Mobile-first، fluid typography، 4 breakpoints (480/768/1024/1280) |
| **انتقالات** | Framer Motion AnimatePresence بين الصفحات (blur + scale + slide) |
| **قائمة موبايل** | قائمة جانبية متحركة مع AnimatePresence + stagger effects |

### 4.2 لوحة التحكم

| الميزة | الوصف |
|--------|-------|
| **تسجيل الدخول** | تصميم نصفين (براند + فورم)، bcrypt، rate limiting، رسائل خطأ عامة |
| **لوحة التحكم** | 4 بطاقات إحصائية، ملخص طلبات (دائري)، آخر 8 طلبات في جدول |
| **إدارة المنتجات** | CRUD كامل: بحث، تصفية حسب القسم، جدول مع صورة، حالة، مميز |
| **إضافة/تعديل منتج** | فورم كامل: اسم (عربي+إنجليزي)، وصف، سعرين، مخزون، قسم، صورة، تاغات |
| **إدارة الطلبات** | كروت طلبات، تغيير حالة (معلق→مؤكد→شحن→توصيل→ملغي)، رابط واتساب |
| **الإعدادات** | اسم المتجر، رقم الواتساب، نص تعريفي |
| **TopBar** | شعار، اسم الأدمن، بريده، Avatar، رابط زيارة المتجر |
| **Sidebar** | 4 روابط مع أيقونات، زر تسجيل خروج |

### 4.3 نظام الحماية

| الطبقة | التفاصيل |
|--------|----------|
| **المصادقة** | NextAuth.js v5، JWT، bcrypt (12 round)، session check في layout |
| **التحقق** | Zod validation لجميع المدخلات (Login, Product, Order) |
| **Middleware** | Security headers (X-Content-Type, X-Frame, X-XSS, Referrer-Policy) |
| **Sanitization** | تنظيف HTML tags من المدخلات |
| **Rate Limiting** | جاهز للتكامل مع Upstash Redis |
| **SQL Injection** | Prisma parameterized queries (حماية تلقائية) |

### 4.4 API Routes

| المسار | الطرق | الوصف |
|--------|-------|-------|
| `/api/auth/[...nextauth]` | GET, POST | NextAuth handler |
| `/api/products` | GET, POST | قائمة المنتجات + إضافة (أدمن فقط) |
| `/api/products/[id]` | GET, PUT, DELETE | تفاصيل + تعديل + حذف منتج |
| `/api/orders` | GET, POST | قائمة الطلبات (أدمن) + إنشاء طلب |
| `/api/settings` | GET, PUT | جلب + تحديث الإعدادات |
